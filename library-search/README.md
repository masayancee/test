# 近くの図書館検索

Calil APIを使用した図書館検索サービス

## 機能

- 📍 現在地から近くの図書館を検索
- 📚 図書館の詳細情報を表示
  - 住所
  - 電話番号
  - ウェブサイトリンク
  - 郵便番号
- 🔄 リアルタイム検索

## 使い方

1. サイトを開く
2. 「📍 現在地を取得」ボタンをクリック
3. ブラウザに位置情報の許可を与える
4. 近くの図書館一覧が表示されます

## 技術

- HTML5
- CSS3
- JavaScript（Geolocation API、Fetch API）
- Calil API v2

## API

[Calil API](https://calil.jp/doc/api_ref.html)を使用

## ブラウザ対応

- Chrome
- Firefox
- Safari
- Edge

注：HTTPSまたはlocalhostでの利用で位置情報が有効です
