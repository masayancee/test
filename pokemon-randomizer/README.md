# ポケモン ランダム表示

PokeAPIを使ったランダムなポケモン表示サイト

## 機能

- 🎲 ボタンをクリックでランダムなポケモンを表示
- 🖼️ ポケモンのイラスト表示
- 📊 ポケモンの基本情報（ID、高さ、重さ）表示
- 🏷️ ポケモンのタイプを色付き表示
- ⚡ PokeAPI v2を使用

## 使い方

1. サイトを開く
2. 「ランダムに表示 🎲」ボタンをクリック
3. ランダムなポケモンが表示されます
4. 何度でもクリックして異なるポケモンを表示可能

## 技術

- HTML5
- CSS3（グラデーション、レスポンシブデザイン）
- JavaScript（Fetch API）
- PokeAPI v2

## API

[PokeAPI v2](https://pokeapi.co/docs/v2)を使用

## ブラウザ対応

- Safari
