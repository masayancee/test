# Simple Alarm App

シンプルで使いやすいアラームアプリケーション

## 機能

- ⏰ 現在時刻のリアルタイム表示
- 🔔 カスタムアラーム時刻の設定
- 🎵 アラーム鳴動時に音声通知
- ⚙️ アラームのキャンセル機能
- 📱 レスポンシブデザイン対応

## 使い方

1. アプリを開く
2. 左側の入力欄でアラーム時刻を設定（時間:分）
3. 「Set Alarm」ボタンをクリック
4. 設定した時刻になるとアラームが鳴ります
5. 「Cancel Alarm」でアラームをキャンセルできます

- Firefox
- Safari
